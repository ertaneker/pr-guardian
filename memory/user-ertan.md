# Ertan — Proje Sahibi / İnsan Operatör

**Rol:** Claude'un erişemediği dış sistemlerde operatörlük yapar.

**Sorumluluklar:**
- GitHub hesap açma/yönetme
- API anahtarı temini (Anthropic, Stripe, vb.)
- Ödeme hesap yönetimi (Stripe, banka)
- Domain kaydı
- Yasal işlemler
- Claude'un yönlendirdiği manuel işlemleri yapma

**How to apply:** Kod, araştırma, pazarlama gibi her şeyi Claude yapacak. GitHub token, API key, ödeme hesabı gibi konularda Ertan'a net talimat ver.
